# OS
- Application - OS - CPU + Memory
- Application은 OS Dependent

# File System
- OS에서 파일을 저장하는 트리구조 저장 체계
- 디렉토리
    - 폴더 또는 디렉토리
    - 파일이나 다른 디렉토리를 포함
- 파일
    - 컴퓨터에서 파일을 저장하는 논리적 단위
    - 확장자로 식별

- 경로
    - 컴퓨터 파일의 고유한 위치
    - 절대 경로
        - 루트 디렉토리부터 파일 위치까지의 경로
    - 상대 경로
        - 현재 있는 디렉토리부터 타겟 위치까지의 경로
        - . 현재 디렉토리
        - .. 이전 디렉토리
