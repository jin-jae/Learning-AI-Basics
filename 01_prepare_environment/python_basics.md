- 컴파일러

| 구분       | 컴파일러                         | 인터프리터                    |
|------------|----------------------------------|-------------------------------|
| 작동방식  | 소스코드를 기계어로 먼저 번역     | 소스 코드를 실행시점에 해석하여 컴퓨터가 처리         |
| 실행 속도  | 빠름                             | 느림                          |
| 사용 예시  | C, C++, Java                     | Python, Ruby, JavaScript      |
| 과정  | Source Code -> Compiler -> Assembler -> CPU  | Source Code -> Interpreter -> CPU   |
